Sobre Lorem

"Lorem" é uma palavra que faz parte do que é comumente chamado de "Lorem Ipsum". O Lorem Ipsum é um texto fictício usado na indústria gráfica e editorial como espaço reservado para o conteúdo de uma página antes que o texto real seja inserido. É utilizado para preencher o espaço em uma página, como um rascunho, enquanto o design gráfico está sendo criado. Isso permite que os designers tenham uma ideia de como o layout ficará quando o conteúdo real for adicionado.

O texto Lorem Ipsum é composto por uma série de palavras em latim que não têm um significado real. A escolha de palavras em latim é intencional, pois a língua é amplamente desconhecida pela maioria das pessoas, o que evita que o texto fictício seja interpretado como conteúdo real. O Lorem Ipsum também é frequentemente utilizado em exemplos de design de página da web, preenchendo espaços de texto em maquetes.

A frase inicial mais comum do Lorem Ipsum é:

"Lorem ipsum dolor sit amet, consectetur adipiscing elit."

Essa frase é seguida por várias linhas de texto fictício sem sentido. O Lorem Ipsum é uma ferramenta útil para designers e editores, pois permite avaliar o layout e o espaço de uma página sem ser distraído pelo conteúdo real durante a fase de design.
